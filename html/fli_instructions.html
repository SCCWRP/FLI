This Family Level Index (FLI) is an educational tool designed to emulate the analysis of the California Streams Condition Index (CSCI), but with less requirements on the input data.

To use this tool, first submit your station cooridinates as a CSV file with columns "StationCode", "Lat", and "Long" (see example). Then, check to make sure that your benthic macroinvertibrate (BMI) data are formatted correctly (see example). Upload the BMI and GIS data, choose a subsampling threshold value, and submit. The primary results should be displayed in the browser. To view other aspects of the results, such as the MMI metric scores or OE capture probabilities, select a different report and press Get Report. This will open the report as a CSV file.
